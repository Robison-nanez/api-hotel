# Api Hotel

Api Hotel

### Prerequisitos

- PHP => 8.4
- PDO del postgres

### Instalacion del proyecto

### Composer
```
composer install
```

### Generar Key
```
php artisan key:generate
```

### Realizar Migracion y Seeder
```
php artisan migrate --seed
```

### Iniciar el servidor
```
php artisan server
```

### Vista de la api con Swagger
```
http://localhost:8000/api/documentation
```
